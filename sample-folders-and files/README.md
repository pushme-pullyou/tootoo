# Sample Test Files

This folder contains sample files for testing the TooToo file browser.

## Contents

- **docs/** — Markdown documents and PDF
- **images/** — PNG, JPG, GIF, BMP, WebP, ICO, and SVG files
- **data/** — JSON, YAML, and CSV files
- **audio/** — WAV audio sample
- **video/** — (add mp4/webm/ogg video files here)
- **projects/** — Nested folders (up to 5 levels deep)
- **misc/** — Plain text, JavaScript, and HTML files

## Purpose

Use these files to verify:

1. Tree view rendering and indentation
2. Folder expand/collapse behavior
3. Auto-display of README.md when opening folders
4. Markdown rendering (headings, lists, code, links, images)
5. Image display (PNG, JPG, GIF, BMP, WebP, ICO, SVG)
6. Audio playback (WAV, MP3, AAC, FLAC)
7. Video playback (MP4, WebM, OGG)
8. PDF viewing
9. HTML iframe rendering
10. Raw/rendered toggle
11. Breadcrumb navigation
12. Filter/search functionality
13. Keyboard navigation
14. Deep nesting support
