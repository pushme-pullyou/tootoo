# TooToo — Tips


## Header

* Click the title to reload — or to reset back to the original repo and clear the current file.
* Click the Octocat icon to open this app's repository on GitHub.
* Click 🌙 to toggle dark mode — your choice is saved across visits.
* Use A− / A+ to adjust font size — helpful on phones and small screens.
* Click the "?" button for the About page: live rate-limit status, repo stats, and the full keyboard-shortcut list.
* Click the "⚙️ Token" button to add a GitHub Personal Access Token — raises the limit from 60 to 5,000 API requests per hour and unlocks private repos.
* The badge beside the buttons shows your remaining GitHub API requests, turning yellow and then red as you run low.

## Sidebar

* Use the filter input to search folder and file names — dashes and spaces are treated alike, so "tootoo why" finds `0-tootoo-why.md`.
* Press Esc in the filter (or click the ✕) to clear it.
* Click "Expand All" to open every folder at once; click again to collapse them all.
* Use your cursor keys to scroll through all the files. Files load the moment you land on them. No need to press Enter or the spacebar.
* Drag the divider between the sidebar and the content to resize it — the width is remembered.

## Content Window

* Click the "View on GitHub" icon to open the current file on GitHub.
* In the breadcrumb trail, the owner, repo, and branch links open GitHub; clicking a folder name scrolls the sidebar to that folder.
* Click the "Rendered" / "Raw" button to toggle between the formatted view and the source — for Markdown, HTML, and SVG.
* Click the "📋 Copy" button to copy the file's text to your clipboard.
* Click the "New Tab" button to open the file on its own — a shareable link for the current file and branch.
* As you browse, the page address updates to the current file, so you can bookmark or share the link directly.

## Keyboard

* `/` — jump to the filter box.
* `\` — jump into the file tree at the current file.
* `↑` / `↓` — move to the previous / next file (it opens as you land).
* `←` / `→` — collapse / expand the focused folder.
* `Home` / `End` — jump to the first / last file.
* `PgUp` / `PgDn` — move about one screen at a time.
