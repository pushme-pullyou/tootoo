# TooToo — Tips



## Header

* Click the title in the header to reload
* Click the Octocat icon to go to the source code on GitHub.
* Click 🌙 to toggle dark mode — your choice is saved across visits
* Use A− / A+ to adjust font size — helpful on phones and small screens
* Click the "?" button in the header for more tips and info about TooToo.
* Click the token button to enter your Personal Access Token.

## Sidebar

* Use the filter input in the sidebar to search folder and file names
* Use your cursor keys to scroll through all the files. Files load the moment you land on them. No need to press Enter or the spacebar.

## Content Window

* Click the "View on GitHub" button to view the file on GitHub.
* Click on the items in the breadcrumb menu Go to their page on GitHub.
* Click the "Rendered" and "Raw" button to toggle the views of the file content.
* Click the "New Tab" button to get a shareable URL for the current file and branch.
