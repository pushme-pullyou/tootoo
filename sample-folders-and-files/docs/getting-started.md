# Getting Started

Welcome to the **sample project**. This guide walks you through setup.

## Prerequisites

- A modern web browser
- Basic knowledge of HTML/CSS/JavaScript

## Installation

No installation needed — just open `index.html` in your browser.

```html
<!DOCTYPE html>
<html>
  <body>
    <h1>Hello World</h1>
  </body>
</html>
```

## Quick Start

1. Clone the repository
2. Open the file browser
3. Navigate the tree view
4. Click any file to view its contents

## Tips

- Use the **filter** to quickly find files
- Press `Arrow` keys to navigate the tree with your keyboard
- Click a folder to auto-display its `README.md`
